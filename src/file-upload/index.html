Styled Native File Inputs

Cross-browser styling for the native file inputs.

See source on GitHub, Index of input patterns

Demo

Demo Details

To style native file inputs, a specific markup pattern is required: 

<div class="file-up" data-file-input>
  <label class="file-up__label" for="file_upload">
    <span class="file-up__label__text">
      Upload
    </span>
    <input type="file" id="file_upload">
  </label>
</div>

The containing div[data-file-input]
