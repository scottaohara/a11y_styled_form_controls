Styled Native Radio Buttons

Cross-browser styling for the native radio buttons.

These custom styles require a specific markup pattern: 

<div class="c-radio">
  <input type="radio" name="my_radios" id="unique_id">
  <label for="unique_id">
    Label text here
  </label>
</div>

The .c-radio container largely serves as the styling hook, and utilizes position: relative; to keep child absolutely positioned elements relative to the container.

The actual native radio button is visually hidden from view. While visually hidden, it is essential for it to remain accessible to screen readers and keyboard users. Otherwise all the native radio button functionality would need to be rewritten in JavaScript.

The label's :before and :after pseudo elements are used to recreated the visible radio button.

The styling of the custom radio button is determined by the state of the actual radio button. :checked, disabled and :focus will all visually change the styling of the custom radio button.
