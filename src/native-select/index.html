Note:

option and optgroup are still notoriously difficult to style in browsers. Often the browser default styles are resistant to any CSS modifications. Though Firefox does give some leeway in how the children of a select can be styled. But even Firefox has its limits, where the optgroup is so limited in styling options that if you require the use of an optgroup, you'd be better off not styling the select.
